# KNNeighborsClassifier Bank

Machine Learning classification project using the K-Nearest Neighbors (KNN) algorithm on Bank Marketing Dataset.

## 📌 Project Overview

This project aims to classify whether a bank customer will subscribe to a term deposit (`yes` or `no`) using the K-Nearest Neighbors (KNN) algorithm.

The dataset contains customer information such as:

- age
- job
- marital status
- education
- balance
- housing loan
- personal loan
- contact type
- campaign information
- previous marketing outcomes

Target prediction:

```bash
y = yes / no
```

## 📂 Dataset

Dataset used:

```bash
bank.csv
```

Example dataset columns:

```bash
age, job, marital, education, balance, housing, loan, duration, campaign, y
```
